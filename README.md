**Mentors:** Vivekanand, Dhanush Sanjay, Pavan Sai.

# Overview

This project explores one of the most famous problems in computer science and optimization — the **Travelling Salesman Problem (TSP)**.

Mentees will learn the mathematical foundations, graph algorithms, heuristics, and optimization strategies used to solve TSP and related routing problems. The project focuses on both theoretical understanding and practical experimentation.
